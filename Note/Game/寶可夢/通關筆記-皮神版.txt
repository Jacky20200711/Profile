# 隊伍配置(根據長期研究，最順的配置如下)
初期 : 尼多王
中期 : 三地鼠 + 雷精靈
後期 : 三地鼠 + 雷精靈 + 急凍鳥 (若不想用神獸，可將急凍鳥改成白海獅)
注意 : 神奇軟糖先做保留，打四天王之前再吃(確保三地鼠60等以上，否則火力會差一些)

PS:有幾隻戰力強大，但不好抓或不好練
> 大食花雖然強，但口呆花要38等才能學會飛葉快刀
> 勇吉拉雖然強，但凱西不好抓(但黃版可以到"玉虹遊戲城"用代幣購買)，且後期沒有活躍機會
> 寶石海星雖強，但海星星很難釣...
> 尼多王不能學會挖洞，中期戰力嚴重不足，要到後期學習地震才比較強
> 嘟嘟利雖然強，但技能/屬性優勢不如三地鼠/勇吉拉
> 四色版只要速度種族64以上，使用"易中要害"其爆擊率幾乎100%(穿山王速度低於64...)

# 冒險預備
1.先到商店買一些精靈球、補血藥、解毒劑(防獨角蟲的毒針)
2.到PM中心左側，或是"進入常磐森林之前的屋子"下方抓"尼多朗"，做為初期的主力
> 打小剛之前必須學會"雙飛踢"(限黃版)

# 第一道館
1.挑戰館主，尼多朗輕鬆輾(雙飛踢)
> 若大岩蛇放"忍耐"，則出皮卡丘用"搖尾巴"降防

2.穿過月見山之前，記得補充藥品、各種解藥
> 在月見山洞窟可以撿到"月之石"與"水槍"，帶著"尼多王"可以輕鬆過山洞

# 第二道館
1.到最上方和NPC交談取得"小火龍"(協助交通，砍樹/怪力/挖洞)
> 一路往右打，會發現研究室，操作電腦幫屋主恢復成人類後可取得"船票"

2.挑戰館主，尼多王輕鬆輾壓

# 第三道館
1.到右側山洞捕捉"地鼠"(19等以上才會挖洞)
> 可以從山洞回到第一道館的城鎮，出洞口後往下，砍掉一棵樹後繼續往下
> 和屋內左邊的人交談可以取得"閃光"(要有10隻以上怪獸他才會給)

2.挑戰館主，三地鼠輕鬆輾壓
> 找到第一個垃圾桶後先存檔，若猜錯就讀檔

3.進入道館上方屋內，聽完最裡面的人講完廢話後取得"自行車兌換卷"
4.返回華藍市兌換自行車，穿過右側的山洞抵達"紫苑鎮"
> 以尼多王的"狂暴"與"泡沫光線"為主力，可以輕鬆過山洞
> 抵達"紫苑鎮"後，先到幽靈塔練一下三地鼠(一路鑽到塔頂)

# 第四道館
1.到購物中心的頂樓買飲料，交給PM中心右邊房子的警衛，即可通往第六道館的城鎮(金黃市)
> 拿不同的種類的飲料，可和販賣機左下的小孩交換(冷凍光束/土石流/三角攻擊)
> 到金黃市PM中心右側某一間屋內，和屋主交談可取得"精神妨礙"

2.到玉虹市PM中心左側大樓，爬到樓頂可以獲得"伊布"
3.到商店購買"雷之石"，進化成雷精靈後學習"十萬伏特"

4.到卡比獸左上方屋內，和女孩交談可以取得"飛天"
5.到遊戲城右下方的屋內，可以撿到"代幣盒子"
6.前往遊戲中心購買230代幣，可到右側交換"凱西"(學習精神妨礙後即可輕鬆訓練)
7.到遊戲中心打火箭隊，打倒最底層的隊員後旁邊會掉落出"電梯卡"
8.搭電梯到頂層並戰勝火箭隊老闆後，旁邊會掉落出"特殊眼鏡"
9.拿到特殊眼鏡後，回到"紫苑鎮"的幽靈塔
> 到頂樓和老頭說話會被傳送到一間房屋內，往前和老頭說話後可以取得"精靈笛"(用來喚醒卡比獸)
> 若沒有拿到笛子，表示背包已經裝滿20個物品

10.從紫苑鎮往下，穿過卡比獸不久，與民宅屋主交談後取得"厲害釣竿"(可用來釣海星星)
11.挑戰館主，三地鼠輕鬆輾壓(此時約40等)

# 第五道館
1.補充道具(超級球/補藥/元氣塊/萬靈通)
2.到狩獵場深處撿拾"金牙"，並到左側屋內和男子交談，取得"衝浪"
3.將金牙交給PM中心右邊民宅內的男子，取得"怪力"
4.挑戰館主，三地鼠輕鬆輾壓(此時約46等，雖然打毒+蟲沒有雙倍，但還是能贏)

# 第六道館
1.到市中心的大樓掃蕩火箭隊，之後才能進入第六道館
2.在5F的下方(只有一人能通過的狹窄走道)可以撿到"鑰匙卡"(用來進出大樓內的任意區域)
3.在3F偏中間的地方使用鑰匙卡進入後，從桌子左下方的瞬移點可以前往基地深處(會看到小茂)
4.打倒火箭隊老大後，往前和禿頭男說話後取得"天王球"
5.挑戰館主，三地鼠輕鬆輾壓(三地鼠此時約56等左右)

# 第七道館
1.前往雙子島捕捉急凍鳥(使用天王球即可)
2.進入道館左邊那棟，在最深處可以撿到第七道館的鑰匙
3.挑戰館主，三地鼠輕鬆輾壓

# 第八道館
1.挑戰館主，急凍鳥輕鬆輾壓

# 挑戰聯盟
> 先將神奇軟糖吃一吃(確保三地鼠60等以上，否則秒不掉迷唇姐/耿鬼/胡地/三合一磁怪/火精靈)
1.第一天王 : 雷精靈
2.第二天王 : 急凍鳥
3.第三天王 : 三地鼠 + 雷精靈(秒大嘴蝠)
4.第四天王 : 雷精靈(秒暴鯉龍/化石翼龍) + 急凍鳥(秒哈克龍/快龍)
5.冠軍小茂 : 急凍鳥 + 三地鼠 + 雷精靈
