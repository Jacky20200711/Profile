# 第一道館
1.先去喇叭牙之塔練功，雖然小鋸鱷的屬性劣勢，但有等級優勢還是可以打
2.挑戰道館，小鋸鱷15等輕鬆打贏
3.到PM中心拿蛋，否則無法前往下個城鎮(會有人擋路)

# 第二道館
1.到做球館接任務，拯救呆呆獸
2.挑戰道館，22等藍鱷輕鬆打贏
3.進入左側森林，趕回大蔥鴨後可得"砍樹"
> 砍樹後，進入右側深處可得"頭撞"

# 第三道館
1.到百貨中心購買"冰凍拳"給藍鱷學習
2.先到上方(35號道路)練功/撿拾挖洞，練到30等進化成"大力鱷"
3.挑戰道館，大力鱷使用"頭撞"輕鬆輾壓
4.進入道館右側的屋內，和女孩交談後取得"灑水器"
5.到廣播塔回答5個問題(是、是、否、是、否)可得"收音機"

# 第四道館
1.到PM中心上方的屋內，戰勝所有藝妓後，和下方紳士交談可得"衝浪"
2.觸發左上方的三神獸劇情
3.挑戰道館，一路咬咬咬輕鬆輾壓

# 第五道館(湛藍市)
1.前往淺蔥市燈塔頂端接任務
2.前往淺蔥市PM中心的左方餐廳，和某人交談後取得"怪力(推石頭)"
3.前往湛藍市取得"秘傳藥"
4.挑戰湛藍市道館，大力鱷隨便打打(準備解催眠，快泳蛙會催眠術!)
5.戰勝道館後，和道館下方閒晃的妹子交談可得"飛天"
6.到最上方觸發水君劇情

# 第六道館(淺蔥市)
1.返回燈塔頂端，以"秘傳藥"治療受傷的電龍
2.挑戰道館，大力鱷隨便打打

# 第七道館
1.解決紅色暴鯉龍事件(因怪異電波導致其發狂)
2.解決火箭隊事件Part1
> 要打倒深處的幹部必須先掌握兩道密碼
> 其中一道密碼要問B3F的女隊員，另一道要問B3F最左側的男隊員
> 打倒幹部後，和旁邊的黑暗鴉交談，取得發電機房的密碼
> 前往發電機房，與阿渡一同拯救被強行關押的頑皮彈

3.挑戰道館，大力鱷隨便打打
4.前往滿金市，解決火箭隊事件Part2
> 前往廣播塔，打倒幹部後取得"地下道鑰匙"
> 前往地下道(開門順序321，若順序亂掉則進出右側樓梯重置)，在最深處和紳士交談取得"廣播塔鑰匙"
> 返回廣播塔解決火箭隊，取得"透明鈴鐺"

5.穿過第七道館右側的冰雪山洞，前往"煙墨市"(途中會撿到"爬上瀑布")

# 第八道館
1.挑戰道館，大力鱷隨便打打(冰凍拳+劈開)
2.前往龍之洞穴，取得徽章
3.返回若葉鎮，和博士交談後取得"天王球"
4.返回湛藍市(第四道館)，前往火燒塔右側靠近水池的民宅捕捉水君

# 挑戰聯盟
1.冠軍之路途中可以撿到"地震"，先不要給大力鱷學(打四天王用不到)
2.挑戰聯盟，大力鱷隨便打打(只有月精靈比較難打)

# 聯盟之後
1.和博士拿船票，前往關都地區
2.有點懶得打，暫時停止闖關
